### 网安学院选修课大观

首先，网安学院开展的选修课基本上以科普各领域经典知识为出发点，总体课程难度设置不会很难，都是基础知识和根据老师不同而相异的前沿知识的科普，定性为**通识性**教学（类似于导论），所以不必畏难。

其次，同学们最关心的考核方式和给分情况，网安的选修课考核一般范式：pre + 期末论文（结合实验），少部分课程有期末考，但不会很难。基本上认真完成课内的作业就能取得不错的分数，没有必要过于焦虑分数的情况。

最后，选修课开展的初衷是满足同学们对不同领域进行自行探索的需要，为后续研究方向的选择提供参考，还是鼓励大家多多选择自己感兴趣的课程进行修读，相信学习这些课程是有所裨益的。不必困囿于分数和优绩的囚笼中而抛弃了最初的兴趣和内心的追求。加油！

> 专业选修共10学分，每门课2学分，需自选5门课程。考虑到大三时间紧任务重，大四面临重要人生选择，如果大二有开设选修课都建议修习，为大三和大四减轻负担。
